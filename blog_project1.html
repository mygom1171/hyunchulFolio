Jeju Travel Management System

  • Hyunchul Eom
  • Java Development
  • Github

여행 계획 작성을 도와주는 프로그램입니다.

[#구현 목표]
 사람들은 고민합니다. 어디로 여행을 갈지, 누구랑 갈지, 여행지 정보는 어디서 찾을지.

[#문제 발생 및 문제 해결]
 더미 데이터를 만들 때, 구분자를 적절히 붙이지 못한 까닭에 코드를 전면 수정해야 하는 문제가 있었습니다.

[#역할]
 프로그램의 전체적인 흐름을 잡고, 여행 일정을 작성하는 부분을 구현했습니다. 또한, 여행 일정을 확인하는 부분을 구현했습니다. '총 일정'과 '날짜 별 세부 일정'을 선택하여 확인할 수 있도록 했습니다. 맡은 부분의 기능 구현을 완료한 후에는, 팀원들이 만들어 온 기능들을 한 데 합치는 역할을 맡았습니다. 하나의 완성된 프로그램을 만든 후, 오류가 발생하는 부분이 없는지 확인하였습니다.

시작하기 전에는 현재 실력으로 맡은 업무들을 해낼 수 있을지 의문이 들었습니다.
차근차근 진행 해 나가면서 구상했던 화면들이 하나 둘 씩 구현되는 모습을 보면서
그러한 걱정은 사라졌습니다. 충분히 생각만 한다면 구현하지 못하는 것은 절대 없다는
생각이 들면서부터 스스로에 대한 확신을 가지게 되었습니다.
함께 해준 팀원분들에게 감사하며, 다음 프로젝트도 최선을 다하겠습니다.
